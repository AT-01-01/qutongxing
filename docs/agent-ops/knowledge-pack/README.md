# 首轮熟悉文档索引

本目录是多 Agent 的首轮熟悉产物，后续需求分派前默认先查这里。

## 文档列表
- `fe-flow.md`：前端页面地图、路由矩阵、页面能力边界。
- `fe-api.md`：前端 API 清单、鉴权时序、重试与错误处理边界。
- `be-auth.md`：认证登录方式矩阵、核心 DTO 契约、风险点。
- `be-activity.md`：活动生命周期、业务接口与副作用。
- `be-data.md`：实体关系与仓储职责。
- `be-platform.md`：安全/限流/异常/跨域现状与风险。
- `runtime.md`：本地联调与构建运行基线。

## 使用方式
- 分派任务前先定位涉及文档，确认“已有行为”与“拟改行为”是否冲突。
- 需求实现后回写文档，保证知识包持续有效。
